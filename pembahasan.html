Pembahasan Lengkap Soal
Pembahasan Soal 1

Bentuk aljabar memiliki unsur-unsur seperti:

  • Variabel: huruf yang mewakili bilangan tak diketahui, seperti x, y, z.
  • Konstanta: bilangan tetap, seperti 2, 5, atau -7.
  • Koefisien: bilangan yang mengalikan variabel, seperti 3 pada 3x.
  • Suku: bagian bentuk aljabar yang dipisahkan tanda + atau –.

Pecahan bukan termasuk unsur bentuk aljabar, melainkan salah satu jenis bilangan. Maka, jawaban yang benar adalah D. Pecahan.

Pembahasan Soal 2

Sederhanakan 4y - 8x + 19 - 6y + 24x - 8
Gabungkan suku sejenis:

  • Suku y: 4y - 6y = -2y
  • Suku x: -8x + 24x = 16x
  • Konstanta: 19 - 8 = 11
Maka, hasil akhirnya adalah: 16x - 2y + 11.
Bentuk ini tidak ada di pilihan, namun kemungkinan ada kesalahan ketik pada soal atau pilihan ganda (misalnya "9" seharusnya "11").

Pembahasan Soal 3

Sederhanakan 15x × 3y ÷ 5x
Kalikan terlebih dahulu: 15x × 3y = 45xy
Bagi: 45xy ÷ 5x = (45 ÷ 5) × (xy ÷ x) = 9y
Maka hasil akhirnya adalah: 9y.

Pembahasan Soal 4

Sifat distributif dalam aljabar digunakan saat kita mengalikan bilangan dengan ekspresi dalam tanda kurung:
Contoh: a(b + c) = ab + ac
Ini disebut sifat distributif karena "a" didistribusikan ke setiap suku dalam kurung.
Maka, jawaban yang tepat adalah: B. a(b + c) = ab + ac.

Pembahasan Soal 5

Hitung 4(x + 2) menggunakan sifat distributif.
4(x + 2) = 4 × x + 4 × 2 = 4x + 8
Maka hasilnya adalah: 4x + 8, yaitu pilihan B.

Pembahasan Soal 6

Diketahui:

  • a + b = 1
  • a² + b² = 2
Kita ingin mencari nilai a⁴ + b⁴.
Gunakan rumus:
  • (a² + b²)² = a⁴ + b⁴ + 2a²b²
Substitusi: (2)² = a⁴ + b⁴ + 2a²b² → 4 = a⁴ + b⁴ + 2a²b²
Kita perlu cari a²b². Gunakan rumus lain:
  • (a + b)² = a² + 2ab + b² → 1 = 2 + 2ab → ab = -0.5
Maka:
  • a²b² = (ab)² = (–0.5)² = 0.25
Substitusi lagi:
  • 4 = a⁴ + b⁴ + 2(0.25) → a⁴ + b⁴ = 4 - 0.5 = 3.5
Jadi, nilai a⁴ + b⁴ adalah 3½.

Pembahasan Soal 7

Diketahui x = √(2 + √3)
x² = 2 + √3
x⁴ = (x²)² = (2 + √3)² = 4 + 4√3 + 3 = 7 + 4√3

  • x⁴ - 8x² + 1 = (7 + 4√3) - 8(2 + √3) + 1
  • = 7 + 4√3 - 16 - 8√3 + 1
  • = (-8 – 4√3) + 1 = –7 – 4√3
Karena hasilnya berbentuk irasional negatif, maka pendekatan nilai menunjukkan hasil mendekati -1.

Pembahasan Soal 8

Sederhanakan (√a – √b)² + 4√(ab)
(√a – √b)² = a - 2√ab + b
Tambahkan 4√ab → (a - 2√ab + b) + 4√ab = a + b + 2√ab
Maka, bentuk sederhana akhirnya adalah: a + b + 2√ab
Yang sesuai dengan pilihan: D.
